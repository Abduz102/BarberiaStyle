<h3>Proyecto integrador</h3>

Para tenerlo en su ordenador nomas tienen que:

2-Abrir la terminal desde la ubicacion donde quieres guardar el repo
3-colocar el comando, git clone URL del repositorio
3-Eso hara una copia de todo el repo 

Hacer cambios para luego devolverlo con las actualizaciones otra vez a github:

1-git add .
2-git commit -m "Mensaje del cambio realizado"
2-git push origin main

Es necesario que hagan estos procesos ya todos son colaboradores por ende pueden hacerlo sin problema alguno
