<h3>Proyecto integrador</h3>

Para tenerlo en su ordenador nomas tienen que:

2-Abrir la terminal desde la ubicacion donde quieres guardar el repo <br>
3-colocar el comando, git clone URL del repositorio <br>
3-Eso hara una copia de todo el repo <br>

Hacer cambios para luego devolverlo con las actualizaciones otra vez a github:<br>

1-git add .<br>
2-git commit -m "Mensaje del cambio realizado"<br>
2-git push origin main<br>
<hr>
Es necesario que hagan estos procesos ya todos son colaboradores por ende pueden hacerlo sin problema alguno
